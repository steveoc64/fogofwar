ActionFront User Manual

Navigating around the App

Getting around the App is simple, fast and consistent. At the top of every screen is the "ActionFront" logo, and clicking on this will always get you back to the main screen :

List Views

On list screens, simply clicking on a row will take you into another screen to view and edit the record that you clicked on.

The Menu Button

Also, in the top right hand corner, there is a "menu" icon to slide in a menu that you can use to jump to any other part in the App.

Add New Record Button

Once you are in a list of records, you can add new records to the list.

Here is an example with the scenarios list - click the "+" button to be able to create a new scenario.

This will take you to another screen to enter the new record details.

Delete Current Record Button

Whenever you are editting a record that you own, you hawe the option of deleting it.

Click on the "-" button on the top of the list to be able to delete it.

Before actually removing the data, the App will always ask you to confirm .. and fill the room with big red warning lights to get you attention to.

Print Page Button

Many of the screens in the App have a "Print" button, which can be used to generate a print-friendly version of the page that you are looking at .. so you can print out Orders of Battle, or Player Briefings for your players.

Click the "Print" button as follows :

Which produces a print-friendly version of the page for hardcopy, or saving to a PDF file.

Action Buttons

Several screens in the App display additional actions that allows you to do things with the current record that you are looking at. Click on any of these "Action Buttons" to go ahead and perform the required action.
